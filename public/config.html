⚙️ Config — ตารางที่เปิดให้ Insert

เลือกว่า API อนุญาตให้ INSERT ข้อมูลเข้าตารางไหนได้บ้าง (ค่าเริ่มต้น: ปิดทุกตาราง)
⚠️ ข้อควรระวัง: INSERT ตรงเข้า database ข้าม business logic ของ Odoo (เลขรันเอกสาร, ฟิลด์คำนวณ, สต๊อก) — เหมาะกับตาราง custom หรือตารางข้อมูลง่าย ๆ เท่านั้น อย่าเปิดตารางเอกสารอย่าง sale_order, account_move, stock_move

วิธีเรียกใช้ Insert API

ตารางต้องถูกเปิดในหน้านี้ก่อน แล้วเรียก:
POST /Odoo/api/odoo/insert/<ชื่อตาราง>
Authorization: Bearer <token>
Content-Type: application/json

{ "data": { "คอลัมน์": "ค่า" } }          // แถวเดียว
{ "data": [ {...}, {...} ] }              // หลายแถว (สูงสุด 500, ทำใน transaction)

เพิ่ม ?dryRun=1 ท้าย URL = ดู SQL ที่จะรันโดยยังไม่ insert จริง
